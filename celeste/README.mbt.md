# celeste

A Celeste-style platformer plugin built on top of `platformer_platform`.

## Highlights

- Single-air dash (`C` or `Left Shift`)
- Wall jump and wall slide
- Hazard spikes
- Dash crystals that refill dash
- Springs that launch the player upward
- Two handcrafted stages

## Run

```bash
moon run cmd/celeste
```

# topdown_dialogue

`topdown_dialogue` builds on [`/Users/zhengyu/Documents/projects/gameparty/topdown_platform`](</Users/zhengyu/Documents/projects/gameparty/topdown_platform>) and adds:

- NPC definitions bound to map spawn tags
- dialogue scripts and line progression
- interaction radius and prompt handling
- dialogue hooks and runtime snapshot
- dialogue panel rendering

It does not add combat, waves, drops, or action AI. Those remain in `topdown_action`.

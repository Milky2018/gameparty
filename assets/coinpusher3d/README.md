# CoinPusher3D Assets

This folder contains visual/audio assets used by `coinpusher3d`.

## License and Usage

- These assets are provided for prototype and non-commercial use inside this repository.
- Do not redistribute these files as a standalone asset pack.
- For commercial use, replace these files with assets you fully own or have explicit commercial licenses for.

## Source Notes

- `textures/table_top.png`, `textures/table_side.png`: adapted from in-repo `pacman3d` textures.
- `textures/coin.png`, `textures/pusher.png`: adapted from in-repo `mooncraft` textures.
- `ui/chest_panel.png`, `ui/hud_bg.png`, `sounds/*`, `music/run_loop.mp3`: adapted from in-repo `thetawave` assets.
- `../shared/fonts/mgs1codec.ttf`: shared in-repo game font asset.
